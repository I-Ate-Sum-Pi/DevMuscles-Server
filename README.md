# DevMuscles-Server

## Routes

|Route|Allowable Methods|
|---|---|
|/users|GET, POST|
- Upon Get Request this will yeild all the users from the database. Expected output is as follows:
```
[
    {
        "id": 1,
        "username": "admin",
        "first_name": "admin",
        "last_name": "admin"
    },
    {
        "id": 12,
        "username": "Akash",
        "first_name": "akash",
        "last_name": "surname"
    },
    ...etc
]
```
